## 📂 Project Structure

Our project is beautifully structured into two main parts:

### 🖥️ Frontend (React)

Located in the `/frontend` folder, our frontend is a React masterpiece, complete with client-side routing via `React Router`.

**Key Dependencies:**
- React `^18.3.1`
- React Router DOM `^6.26.2`
- Axios `^1.7.7` for seamless HTTP requests
- Auth0 React `^2.2.4` for secure authentication

**Folder Structure:**
- `/components`: Reusable UI components for a polished look.
- `/screenPages`: Unique pages for user actions like signup and login.

### ⚙️ Backend (Node.js & Express)

The `/Backend` folder houses our powerful backend, built using `Node.js` and `Express`.

**Key Dependencies:**
- Express `^4.21.0`
- Mongoose `^8.6.2` for elegant MongoDB interactions
- JWT `^9.0.2` for secure authentication tokens
- Bcrypt `^5.1.1` for safe password handling
- Cors `^2.8.5` for smooth Cross-Origin Resource Sharing

**Folder Structure:**
- `/Routers`: Defines our API routes.
- `/models`: Mongoose schemas that give life to our database.
- `/middleware`: Handles authentication and other essential tasks.
- `/config`: Houses configuration files, including environment variables.

## 📦 Installation

Ready to embark on this journey? Follow these steps to set up the project locally:

### 🖥️ Frontend Setup
1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```

### ⚙️ Backend Setup
1. Navigate to the `Backend` directory:
   ```bash
   cd Backend
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Launch the backend server:
   ```bash
   npm run start
   ```
